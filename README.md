# Owl Top

## About

Web app for observing independent rating of schools and courses.

## Tech

- Next.js
- React.js
- TypeScript
- CSS Modules
- ESLint
- Stylelint
- Framer Motion animations

## Run project locally

```bash
git clone https://github.com/artemshchirov/next-owl-top.git
cd next-owl-top/
yarn install
yarn build
yarn start
```
